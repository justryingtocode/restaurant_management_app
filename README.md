# Restaurant-Angular-App

Welcome to the Project! This project provides a simple reataurant management functionality with username and password authentication.

## Getting Started

These instructions will help you get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Node.js (v18 or higher)
- Nvm is highly recommended.
- npm (v6 or higher)
- MySQL database

### Installation

1. **Clone the repository**:
2. **Install the dependencies**:
	npm install, npm install --legacy-peer-deps
3. **Setup the database**:
    - Import the provided SQL file to your MySQL database. This file will create the necessary tables and insert some initial data.
    - Use a MySQL client or the command line to import the database:

4. **Configure the database connection**:
    - Update the `env` file with your MySQL database credentials.

5. **Run the application**:
	Note: Both front-end and back-end code is in this repo. YOu will have to run those projects separately.
	- npm run start

![image](https://github.com/justryingtocode/restaurant_management_app/assets/51942207/af3fa154-834e-4079-af76-2a22c144826e)

6. ### Admin Login
	After project is running in browser you can admin login with
- **Username**: `admin`
- **Password**: `1234`
